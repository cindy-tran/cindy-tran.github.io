Technologies Used: Javascript, HTML, CSS, React, Node.js, Express, MongoDB

Group project for UCI's Next Generations Search System project course. This is a web application that helps users locate events on the UCI Campus. Users can search events by time, location, and category, and leave comments on those events.
My role in this project was to design and develop the user interface. I consumed the application's REST API along with Google Maps API to display and reflect the data in the database.

Check out the project here: Eventfeeler
